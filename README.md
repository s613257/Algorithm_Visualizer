# 演算法劇場

##### 開發期程： 2025/8~2025/9
#### 本工具特色為使用動畫呈現方式，讓使用者更加理解演算法的邏輯，並且隨時可查看重點筆記。

### 1.功能介紹：
* 排序：分為氣泡、插入、選擇、快速、合併、堆積排序法。
* 搜尋：針對廣度、深度優先搜尋法進行迷宮遊戲。
* 分類與分群：對資料進行分類與分群分析並使用散佈圖呈現。

### 2.使用技術：
* 後端：Python
* 前端：Html、CSS、JavaScript
